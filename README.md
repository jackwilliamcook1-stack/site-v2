# Jack Cook portfolio site

Static, five pages, no build step. Works on GitHub Pages as-is.

## Files you need to add

Drop these into /assets with these exact names:
- assets/econ66-paper.pdf          (your Econ 66 final paper)
- assets/callaway-topgolf-deck.pdf (the Topgolf/Callaway presentation)
- assets/jack-cook-resume.pdf      (your resume)

Photos into /assets/photos with these exact names:
- fishing.jpg
- skiing.jpg
- music.jpg

(4:5 portrait crops look best. Any size; they're displayed ~300px wide.)

## Hosting free on GitHub Pages

1. Create a public repo named jackcook.github.io (or any name).
2. Push these files to the repo root.
3. Settings > Pages > Source: deploy from branch, main, / (root).
4. Site is live at https://<username>.github.io (or /<repo-name>).

Custom domain later: buy one (~$12/yr), add it under Settings > Pages.
